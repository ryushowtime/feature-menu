01 / 概述

什么是
Feature Menu?

一款专为 Claude Code 用户打造的本地工具，帮助你轻松管理和浏览所有已安装的技能、代理和命令。

  • 完全本地运行，数据永不离开你的电脑
  • 自动检测 Claude Code 安装目录
  • 一键启动，打开浏览器即可使用

04 / 智能路径检测

自动检测
Claude Code 目录

无需手动配置，首次使用自动检测你的 Claude Code 安装目录。

  • 环境变量 CLAUDE_CODE_ROOT
  • ~/.claude
  • ~/everything-claude-code
  • 向上自动搜索
// 自动检测优先级

1. CLAUDE_CODE_ROOT 环境变量
2. ~/.claude
3. ~/everything-claude-code
4. ~/Claude Code

08 / 为什么选择

为什么需要
Feature Menu?

  • 找不到技能？ — 安装太多，记不住每个技能的用途
  • 不知道用什么？ — 不确定哪个技能适合当前任务
  • 切换麻烦？ — 需要手动翻找目录和文档

Feature Menu 解决这一切

集中管理、可视化浏览、智能搜索、收藏最近使用 —— 让你的技能库井井有条。
